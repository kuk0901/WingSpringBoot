## UI 디자인 feedback

- 관리자

  ```
  - 판매 카드 현황: 카드 추가될 때를 대해 구분할 수 있는 칼럼 추가


  - 카드 상세

    - 혜택을 간결하게 작성해 사용자가 가계부에서 작설할 때 카테고리와 카드가 매칭됐을 때
      1:1 매칭 후 할인된 금앨을 결제 내역에 띄워줌

    - 할인은 기본 할인으로 통합, 자세한 할인율은 5%로 통합


  - 카테고리 현황

    - 작성된 카테고리 조회, 카테고리를 등록할 수 있는 버튼 유지

    - 수입과 지출 전체 내역을 조회할 수 있는 버튼 생성

      - +인 카테고리들의 합산 금액 조회(년)

      - -인 카테고리들의 합산 금액 조회(년)


  - 카테고리 상세: 합산 금액 부분 삭제
  ```
